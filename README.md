# Salesforce Complaint & Case Tracking System

## Overview
A Salesforce-based Case Management application that allows customers to submit complaints and receive automated status updates.

## Features
- Complaint submission using Lightning Web Components
- Case creation via Apex controller
- Automated email notification on status change
- Trigger handler pattern
- Apex test classes with coverage

## Tech Stack
- Salesforce Apex
- Lightning Web Components (LWC)
- SOQL
- Case Object
- Apex Triggers

## Architecture
LWC → Apex Controller → Case → Trigger → Email

## Status
In active development as part of Salesforce learning.
